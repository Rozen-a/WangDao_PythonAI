## 学习内容

### Day2 - Python基础入门
- 字符串插值（f-string、str.format()、%格式化）
- 数据类型（整型、浮点型、布尔型、字符串）
- 运算符（算术、比较、逻辑运算符）
- if-elif-else条件判断
- while/for循环
- range函数
- 三目运算符

### Day3 - 数据结构基础
- 字符串操作（切片、常用方法）
- 列表（增删查改、排序、反转）
- 元组（不可变序列）
- 字典（键值对存储、常用方法）
- 集合（去重、交集、并集、差集）
- 公共方法（len()、max()、min()、sorted()、enumerate()）
- 推导式（列表、字典、集合推导式）

### Day4 - 函数编程
- 函数基础（定义、调用、参数传递）
- 全局变量与局部变量（global关键字）
- 函数参数（位置参数、关键字参数、缺省参数、多值参数）
- 函数返回值（单值返回、多值返回、拆包）
- 函数作为参数（高阶函数）
- 递归函数
- 类与对象入门

### Day5 - 面向对象编程
- 封装（私有属性和私有方法）
- 继承（单继承、多重继承）
- 多态（方法重写）
- 实战案例（摆放家具、动物类设计）

### Day6 - 高级特性
- 类属性与类方法
- 静态方法
- 单例模式
- 异常处理（try-except、自定义异常、异常传递）
- 文件操作（文件读写、目录操作）
- 模块与包（模块导入、包的使用）
- 深拷贝与浅拷贝

### Day7 - 进阶主题
- 正则表达式（模式匹配、字符串处理）
- 二叉树（数据结构基础）
- 时间复杂度（算法效率分析）
- sorted与sort（排序方法）
